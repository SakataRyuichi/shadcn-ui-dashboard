# サイドメニュー要件

## サイト名

- **Commune Engage**

## メニュー構成

### ホーム
- 展開なし（単一リンク）
- `/` に遷移
- 表示中は primary カラーでアクティブ表示

### ブランド
- ブランド登録: `/brands/register`
- ブランド一覧: `/brands`

### ユーザー
- ユーザー登録: `/users/register`
- ユーザー一覧: `/users`

### その他
- スクリプト: `/scripts`
- 利用規約: `/terms`

## メニュー下部（フッター）

サイドメニューの下部に以下を表示する機能：

- ユーザーアイコン
- ユーザー名
- メールアドレス

## 技術スタック

- shadcn/ui をベースとする
